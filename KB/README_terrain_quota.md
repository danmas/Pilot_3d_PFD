# Система квот Mapbox

**Дата:** 2026-06-10
**Статус:** Реализовано ✅
**Автор:** Hermes Agent
**Назначение:** Документация системы отслеживания и лимитирования запросов к Mapbox API
**Актуализация:** 2026-06-10

---

## Обзор

Mapbox предоставляет **50,000 бесплатных запросов в месяц** на тарифный период. После превышения — блокировка или плата за каждый запрос.

Система квот:
- Считает каждый запрос к Mapbox (DEM + Satellite отдельно)
- Хранит счётчик на диске (переживает перезапуск сервера)
- Останавливает загрузку при достижении 90% лимита (45,000)
- Сбрасывается автоматически 1-го числа каждого месяца

## Файл квоты

**Путь:** `cache/terrain-quota.json`

```json
{
  "month": "2026-06",
  "dem": 47,
  "sat": 0,
  "total": 47
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `month` | string | Текущий месяц в формате `YYYY-MM` |
| `dem` | number | Количество DEM запросов за месяц |
| `sat` | number | Количество Satellite запросов за месяц |
| `total` | number | Общее количество запросов (dem + sat) |

## API endpoint

### GET /api/terrain/quota

**Ответ:**

```json
{
  "month": "2026-06",
  "dem": 47,
  "sat": 0,
  "total": 47,
  "limit": 50000,
  "remaining": 49953,
  "resetDate": "2026-07-01"
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| `month` | string | Текущий месяц |
| `dem` | number | Запросов DEM |
| `sat` | number | Запросов Satellite |
| `total` | number | Всего запросов |
| `limit` | number | Лимит (всегда 50000) |
| `remaining` | number | Осталось запросов |
| `resetDate` | string | Дата сброса (1-е число след. месяца) |

## Алгоритм работы

### Инициализация (loadQuota)

```
1. Проверить cache/terrain-quota.json
2. Если файл есть:
   └─ Прочитать JSON
   └─ month == currentMonth?
        да → использовать
        нет → сбросить { month: current, dem: 0, sat: 0, total: 0 }
3. Если файла нет → создать новый счётчик
```

### Инкремент (incrementQuota)

```
1. month == currentMonth?
     нет → сбросить счётчик
     да → продолжить
2. type == 'dem' → quota.dem++
   type == 'sat' → quota.sat++
3. quota.total++
4. Сохранить файл
```

### Проверка перед запросом к Mapbox

```
1. quota.total >= 45000?
     да → 429 Too Many Requests (блокировка)
     нет → разрешить запрос
```

## Логика сброса

Сброс происходит автоматически 1-го числа каждого месяца:

```typescript
function currentMonth() {
  const now = new Date();
  return `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}`;
}
```

При каждом обращении к quota (в т.ч. `GET /api/terrain/quota`) проверяется текущий месяц. Если месяц сменился — счётчик обнуляется.

## Безопасный буфер

| Параметр | Значение |
|----------|----------|
| Лимит Mapbox | 50,000 |
| Порог остановки | **45,000** (90%) |
| Буфер | **5,000** запросов (10%) |

Зачем буфер:
- Race condition при параллельных запросах (6 concurrent)
- Запросы в полёте не блокируются
- Время на реакцию (отключить terrain, добавить токен)

## Как проверить текущую квоту

```bash
# Через API
curl http://localhost:3410/api/terrain/quota

# Прямое чтение файла
cat cache/terrain-quota.json

# Через браузер
http://87.251.79.241:3410/api/terrain/quota
```

## Поведение при разных состояниях

| Состояние | Что происходит |
|-----------|----------------|
| Квота в норме (< 45k) | Нормальная загрузка, запросы к Mapbox, кэширование |
| Почти кончилась (≥ 45k) | 429 для новых запросов, terrain отображается из кэша |
| Новый месяц | Счётчик автоматически сбрасывается |
| Удалён файл квоты | Создаётся заново с нуля |
| Сервер перезапущен | Квота восстанавливается из файла |

## Тестирование

```bash
# Получить квоту
curl -s http://localhost:3410/api/terrain/quota | jq

# Сбросить квоту (для теста)
rm cache/terrain-quota.json && pm2 restart pilot-3d-pfd

# Эмуляция исчерпания (изменить вручную)
echo '{"month":"2026-06","dem":44980,"sat":20,"total":45000}' > cache/terrain-quota.json
```
